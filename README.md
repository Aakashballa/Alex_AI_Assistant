# 🤖 Jarvis AI Assistant

A personal AI voice assistant inspired by **J.A.R.V.I.S (Iron Man)** — capable of understanding voice commands, responding intelligently, and operating fully offline using speech recognition and LLM integration.

---

## ⚡ Features

- 🎤 Wake word detection ("Alex")
- 🧠 Offline speech recognition using **VOSK**
- 🔊 Text-to-speech via **pyttsx3**
- 🖥️ Interactive GUI with waveform & avatar
- 💬 AI-powered responses using **KoboldCpp**
- ⛔ Stop button to interrupt active responses
- 🔒 Runs fully offline (except initial model download)

---

## 🛠️ Tech Stack

- Python
- VOSK (Speech Recognition)
- pyttsx3 (Text-to-Speech)
- SpeechRecognition
- KoboldCpp (LLM integration)
- Tkinter (GUI)
